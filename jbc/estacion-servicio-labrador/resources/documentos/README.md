# Documentos publicados

Coloque aqui los documentos que se publicaran con el sitio.

Para mostrar el reporte de calidad del resultado fotogrametrico, agregue el archivo a esta carpeta y registre una entrada en `manifest.json`.

Ejemplo:

```json
{
	"documents": [
		{
			"title": "Reporte de calidad de resultado fotogrametrico",
			"type": "quality-report",
			"file": "reporte-calidad-resultado-fotogrametrico.pdf",
			"date": "2026-06-17"
		}
	]
}
```
